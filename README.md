
## Prerequisites

- Python 3.12
- uv
- Dagster


## Setup

1. Install all the dependencies

```shell
uv pip install -e ".[dev]"
```


## Commands

- Create a brand-new code location

```shell
dagster project scaffold --name <brand-new-code-location>
```
